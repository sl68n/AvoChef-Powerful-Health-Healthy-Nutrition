AvoChef is a mobile and web-based healthy food platform designed to connect users with verified healthy restaurants.  
Unlike traditional food delivery apps, AvoChef focuses on **Nutrition First**, offering calorie-based filtering, verified nutrition data, and personalized meal planning.

---

## Project Purpose

Maintaining a balanced diet is challenging due to:
- Lack of time
- Limited access to accurate nutritional information
- Difficulty finding calorie-specific meal plans

AvoChef bridges this gap by providing:
- Verified nutritional data
- Calorie-based meal filtering
- Personalized diet customization
- Seamless integration with restaurant partners

---

## Project Objectives

- Conduct nutrition market analysis
- Build user-friendly UI/UX
- Develop secure front-end & back-end systems
- Establish a verified "Healthy Meals Database"
- Integrate APIs between app, cloud servers, and restaurants
- Perform performance and load testing

## Work Breakdown Structure (WBS)

1. Initiation (10 Days)
   - Vision & Market Study

2. Planning (22 Days)
   - Requirements Gathering
   - WBS Definition

3. Development (68 Days)
   - Design
   - Coding
   - API Integration

4. Monitoring & Controlling (63.75 Days)
   - Testing
   - Database Verification
   - Support Setup

5. Closure (12 Days)
   - Documentation
   - Evaluation
   - Archiving

*Note:*  
Although total phase durations sum to 175.75 days, the actual project duration is 143.75 days due to parallel task execution (Front-end & Back-end running simultaneously).
